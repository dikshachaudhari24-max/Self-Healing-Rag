# FairHire Backend

FastAPI backend skeleton for FairHire.

## Structure
- `app/api`: route handlers
- `app/core`: configuration and environment settings
- `app/models`: request and response schemas
- `app/repositories`: data access layers
- `app/services`: transcription, analysis, RAG, and reporting services

## Run later
1. Create a virtual environment.
2. Install backend dependencies from `pyproject.toml`.
3. Start the app with Uvicorn once the services are configured.
